About

Get to know me better

Who am I.

I am a bilingual and multicultural developer with a BA from University of California Davis; currently living in California. I have a passion for clean, intuitive interfaces and organized, reusable code. I keep up to date with the latest UI trends, and make my client's visions into a reality.

What I Do.

I have been doing freelance web development, graphic design and UI design for 4 years, mainly in the sectors of education and tourism. Developed responsive, lightweight front ends for websites using Bootstrap; as well as Angular.js, React.js javascript frameworks and the Adobe Creative Suite.

My Mission.

In a world where technology is rapidly changing, many users are lost because they are not 'tech savy' enough. I strive to change that. It is not the users who should be 'savy', it is the designers and developers that should be making intuitive products.

Portfolio

Tabarca

I made to advertise a rental home in Tabarca (the smallest inhabitted island in the Mediterranean). It was made using HTML, CSS and JavaScript using Bootstrap as a framework. This service included not only the website but a Facebook and Google Adwords campaign.

IE in Ethiopia

I made this for a program the IE Business Institute was offering as part of it's MBA. Graduate students were sent to Ethiopia to investigate promising local businesses, and they would then select a few to invest in, as a way of stimulating the Ethiopian economy. It was made using HTML, CSS and JavaScript using Bootstrap as a framework.

Be Fluent Online

I made this for an ESL business in Madrid. It was made using HTML, CSS and JavaScript using Bootstrap as a framework. The logo was not made by me, but by a designer who was hired seperately.

OCP Equity Partners
